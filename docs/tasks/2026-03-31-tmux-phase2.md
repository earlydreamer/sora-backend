**Issue**: #5
**상태**: codex-in-progress
**담당**: Codex

## 목표

5개 helper script (`spawn-worker`, `list-workers`, `capture-worker`, `mark-worker`, `recover-session`)를 Bash로 구현하고, `spawn-worker`에 `--auto-start` 옵션을 추가해 tmux send-keys 기반 자동 트리거를 실현한다.

## 범위

**신규 생성:**
- `scripts/orchestrator/lib.sh` — 공통 함수 라이브러리
- `scripts/orchestrator/spawn-worker` — 워커 생성 + 초기 명령 주입
- `scripts/orchestrator/list-workers` — 워커 상태 조회
- `scripts/orchestrator/capture-worker` — 출력 캡처
- `scripts/orchestrator/mark-worker` — 상태 기록
- `scripts/orchestrator/recover-session` — 세션 복구
- `.orchestrator/.gitkeep` — 디렉터리 유지

**수정:**
- `docs/operations/tmux-orchestration.md` — helper script 사용 예시 추가
- `docs/current.md` — 2단계 완료 상태 반영
- `.gitignore` — 이미 추가됨 (1단계)
